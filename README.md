## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/sadia-portfolio.git
```

Go into the project folder

```bash
cd sadia-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
